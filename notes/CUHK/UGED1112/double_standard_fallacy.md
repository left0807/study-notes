# 普羅塔哥拉斯雙重標準謬誤 (Protagoras - Double Standard Fallacy)

**背景**：
古希臘辯士普羅塔哥拉斯（Protagoras）與徒弟訂下合約：
- 徒弟先學習辯論術
- 畢業後徒弟打贏第一場官司時才需支付學費
- 若徒弟一直不打官司，則永遠不用付學費

徒弟學成後卻遲遲不打官司，也不付學費。普羅塔哥拉斯於是控告徒弟。

---

## 雙方論證

### 師父的論證 (Protagoras' Argument)
**前提 1**（從法庭判決角度）：  
如果我勝訴 → 徒弟要依判決交學費

**前提 2**（從合約約定角度）：  
如果徒弟勝訴 → 徒弟打贏第一場官司，依約要交學費

**結論**：  
無論誰勝訴，徒弟都要交學費。

```
形式：
如果 P（師父勝訴）則 Q（徒弟交學費）
如果 R（徒弟勝訴）則 Q（徒弟交學費）
——————————————————
Q（徒弟交學費）
```

---

### 徒弟的論證 (Student's Argument)
**前提 1**（從法庭判決角度）：  
如果我勝訴 → 依判決不用交學費

**前提 2**（從合約約定角度）：  
如果我敗訴 → 尚未打贏第一場官司，依約不用交學費

**結論**：  
無論誰勝訴，徒弟都不用交學費。

```
形式：
如果 R（徒弟勝訴）則 ¬Q（不交學費）
如果 P（師父勝訴）則 ¬Q（不交學費）
——————————————————
¬Q（不交學費）
```

---

## 問題分析：雙重標準謬誤

雙方都採用了**不一致的標準**來解釋合約與判決的關係：

1. **師父的矛盾**：
   - 當自己勝訴時，主張**法庭判決優先**
   - 當徒弟勝訴時，卻主張**合約條款優先**

2. **徒弟的矛盾**：
   - 當自己勝訴時，主張**法庭判決優先**
   - 當自己敗訴時，卻主張**合約條款優先**
